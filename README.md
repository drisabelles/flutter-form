## ⚙️ Executando <a name="running"></a>

```bash

# Baixe este repositório ou clone pelo Git usando o comando:
$ git clone https://github.com/drisabelles/flutter-form.git

# Acesse a pasta do projeto
$ cd flutter-form

# Instale as dependências
$ dart pub get

# Inicie o projeto
$ flutter run

```
